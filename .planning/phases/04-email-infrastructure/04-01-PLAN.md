   - Export `validateMx(email: string): Promise<boolean>`
   - Use `dns.promises.resolveMx()` from Node.js built-in
   - Extract domain after @, return false if no @ or empty domain
   - Catch all DNS errors (ENODATA, ENOTFOUND, ETIMEOUT) and return false

   - Export `getDailyCount(): Promise<number>` — queries email_events where status='sent' and sent_at >= today (UTC midnight). Use createClient from @/lib/supabase/server.

   - Export `DAILY_CAP = 45` and `SEND_SPACING_MS = 90_000` as named constants

3. Create `lib/email/unsubscribe-token.ts`:

   - Export `verifyUnsubscribeToken(email: string, leadId: string, token: string): boolean` — timing-safe comparison via `crypto.timingSafeEqual` (convert both to Buffer, return false on length mismatch)

   - Configure with `service: 'gmail'`, `auth.type: 'OAuth2'`, read GMAIL_SENDER_EMAIL, GMAIL_CLIENT_ID, GMAIL_CLIENT_SECRET, GMAIL_REFRESH_TOKEN from env
   - Throw descriptive error if any env var is missing
